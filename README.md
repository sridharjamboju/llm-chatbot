# Local AI Chatbot using Ollama

This project implements a modular AI chatbot using a locally hosted Large Language Model (LLM).

## Features
- Local LLM inference using Ollama
- Conversation memory support
- Modular Python architecture
- Error handling and API communication

## Technologies Used
- Python
- Ollama
- Qwen2.5 1.5B Model

## Project Structure

llm-chatbot
├── main.py
├── config.py
├── memory_manager.py
├── prompt_builder.py
└── ollama_client.py

## How to Run

1. Start Ollama.
2. Navigate to the project folder:

cd D:\AI-LAB\projects\llm-chatbot

3. Run the chatbot:
python main.py
